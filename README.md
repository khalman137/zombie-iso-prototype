# NewEngine 2.5D v1.9.5

Тест импорта модульного персонажа из самостоятельного ZIP-архива.

## Добавлено

- `characters/Character_Base_Male_01_Test.zip` загружается игрой напрямую;
- движок читает `manifest.json` внутри архива;
- персонаж собирается из слоёв Legs, Torso, ArmLeft, ArmRight и Head;
- Idle и Walk переключаются автоматически;
- кадры Walk читаются из горизонтальных sprite sheet;
- единый pivot берётся из manifest;
- текущий тестовый набор содержит только South и East, поэтому отсутствующие направления временно используют ближайший доступный вариант.

Запускать через HTTP/HTTPS-сервер.
