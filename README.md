# NewEngine 2.5D v1.7

Тестовая сборка новой архитектуры.

## Изменения v1.7

- левый стик задаёт движение и направление взгляда, если правый стик отпущен;
- правый стик временно управляет взглядом независимо от движения;
- чтение объектов из `zN_objects.png`;
- тестовые кровати, шкафы, холодильники, тумбы и раковины;
- коллизии с твёрдой мебелью;
- единое действие `E` / кнопка «Действие»;
- подсветка ближайшего интерактивного объекта;
- тестовое окно контейнера и профили будущего лута;
- полные стены больше не становятся прозрачными;
- стены текущей комнаты со стороны камеры заменяются метровым срезом;
- соседние комнаты сохраняют полные стены;
- тестовый сектор зрения персонажа;
- стены и закрытые двери блокируют прямую видимость;
- движущиеся тестовые цели исчезают вне сектора зрения.

## Структура

- `index.html`
- `maps/test_map.zip`
- `templates/*.zip`

Архивы карт и шаблонов читаются игрой напрямую и не требуют распаковки.

## Запуск

Запускайте через HTTP/HTTPS, например GitHub Pages или raw.githack. Открытие через `file://` может блокировать загрузку архивов.
