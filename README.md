# NewEngine 2.5D v1.9.4

Исправлена система предметов мира.

- Предметы на земле являются отдельными ItemInstance с уникальным UID.
- Выбрасывание меняет владельца предмета на Ground и записывает координаты/этаж.
- Предмет остаётся в точке выбрасывания.
- Кнопка «Бросить» добавлена прямо в строку предмета при просмотре пола.
- Перенос в список «Предметы рядом на полу» больше не записывает предмет во временный массив.
- Ближняя видимость 360° в радиусе 1 м сохранена.

Запускать через HTTP/HTTPS, не через file://.
