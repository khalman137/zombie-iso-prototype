# NewEngine 2.5D v1.9 test

Тестовая версия новой архитектуры.

## Добавлено
- кликабельная кнопка действия между стиками;
- инвентарь игрока: 30 кг и 36 ячеек;
- стаки предметов;
- контейнеры с детерминированным тестовым лутом;
- перенос по одному предмету между игроком и контейнером;
- выбрасывание предметов на пол;
- просмотр предметов на полу через кнопку «Инвентарь»;
- блокировка движения при открытом интерфейсе.

## Управление
- левый стик / WASD: движение;
- правый стик: направление взгляда;
- центральная кнопка / E: действие;
- «Инвентарь» / I: открыть инвентарь и предметы пола.

## Запуск
Запускать через HTTP/HTTPS. `file://` может блокировать загрузку ZIP-файлов.
