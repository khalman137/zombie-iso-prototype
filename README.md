# NewEngine 2.5D v1.9.6

Тест каталога и выбора базовых персонажей.

## Структура

```text
characters/
  characters.json
  Character_Base_Male_01.zip
```

## Что изменено

- основной архив переименован в `Character_Base_Male_01.zip`;
- движок больше не загружает персонажа по жёстко прописанному имени файла;
- сначала читается `characters/characters.json`;
- персонаж выбирается по постоянному `id`;
- выбранный `id` сохраняется в `localStorage` как `selected_character_id`;
- при отсутствии или ошибке выбора используется `default_character`;
- ID внутри архива проверяется на совпадение с записью каталога;
- система готова к добавлению следующих базовых тел без изменения загрузчика.

Запускать через HTTP/HTTPS-сервер.
